## Primeros pasos para comenzar a usar el sistema

¡Felicidades!, ya has logrado iniciar sesión en el sistema con tu cuenta `SUPER ADMIN`.
Ahora debes gestionar unas cuantas cosas antes de comenzar a usar el sistema.

### Paso 1: Agrega empleados

Puedes agregar empleados con distinto rol, como `ADMIN`, `EMPLEADO` y `CAJERO`. Cada uno tiene sus propias responsabilidades y permisos. Tienes mas información en la sección de [Empleados](/ayuda/empleados).

### Paso 2: Crear una categoría

Para organizar tus productos puedes crear categorías dependiendo de su tipo. **Es obligatorio crear al menos una categoría**, ya que la creación de productos lo requiere. Tienes mas información en la sección de [Categorías](/ayuda/categorias).

### Paso 3: Crear una subcategoría

Si quieres tener un mayor control sobre tus productos puedes crear subcategorías, es importante saber que **una subcategoría depende de una categoría previamente creada**. Tienes mas información en la sección de [Subcategorías](/ayuda/subcategorias).

### Paso 4: Crear una unidad de medida

Puedes crear unidades de medida para tus productos, tales como `Kg`, `g`, `L`, `ml`, `unidad`, etc. Al igual que las categorías, **es obligatorio crear al menos una unidad de medida**. Tienes mas información en la sección de [Unidades de medida](/ayuda/unidades-de-medida).

### Paso 5: Crear un producto

¡Lo lograste! Haz configurado todo lo necesario para comenzar a usar el sistema. Ahora si puedes empezar a crear productos dirigiendote a la sección de `Inventario`, con esto **podras empezar a realizar las operaciones de venta**.

### IMPORTANTE

Esta es una pequeña introducción al sistema, faltan muchas mas funcionalidades que puedes explorar en la sección de [Ayuda](/ayuda) si quieres aprovechar al máximo todo el potencial del sistema.
